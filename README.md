# 默记 MoCard

> *"像猫咪舔毛一样，每天温柔地复习一遍~"* 🐾

一只帮你记住所有知识点的小猫卡片应用，专为考研党打造。

---

## 它能做什么喵？

```
  ╭──────────────────────────────────╮
  │  📦 卡组管理    创建/编辑/删除/分类   │
  │  🃏 智能卡片    文字+图片 双面卡片    │
  │  🧠 科学复习    SM-2 间隔重复算法     │
  │  🔍 全局搜索    关键字+标签+筛选      │
  │  📊 学习统计    趋势/进度/连续天数     │
  │  ⭐ 标记系统    收藏/重点/易错        │
  ╰──────────────────────────────────╯
```

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/your-name/mocard.git
cd mocard

# 2. 安装依赖
flutter pub get

# 3. 生成数据库代码
dart run build_runner build --delete-conflicting-outputs

# 4. 跑起来！
flutter run
```

## 技术栈

| 喵~ | 用了什么 | 干什么的 |
|-----|---------|---------|
| 🏗️ | Flutter 3.24+ | 跨平台 UI |
| 📦 | Riverpod | 状态管理 |
| 🗄️ | Isar | 本地数据库 |
| 🧭 | GoRouter | 页面路由 |
| 🧠 | SM-2 | 间隔重复算法 |

## 项目结构

```
lib/
├── core/           # 常量、主题、路由、工具
├── data/           # 数据源、模型、仓库实现
├── domain/         # 实体、抽象仓库、服务接口
└── presentation/   # 页面、组件、状态管理
```

整洁的三层架构，像猫咪理毛一样一丝不苟 ✨

## 架构预留

这只小猫还在长大，已经为未来留好了位置：

- **RemoteDatasource** — 云端数据源接口，随时接入后端
- **SyncService** — 同步服务接口，多设备不怕丢
- **MediaStorageService** — 存储抽象层，本地→云端一键切换
- **ReviewScheduler** — 算法可替换，SM-2 → FSRS 无缝升级

## 环境要求

- Flutter `>= 3.24.0`
- Dart `>= 3.5.0`
- Android SDK（minSdk 跟随 Flutter 默认）
- Java 17
